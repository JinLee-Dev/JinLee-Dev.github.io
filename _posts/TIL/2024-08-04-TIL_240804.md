* int 범위를 넘는 큰수에 대한 계산
    * 보통은 큰 수 자체를 출력하기보단, 나눈 값을 요청하는 문제가 많다.
    * 수학이 제일 어려움
    * 나머지 연산 분배법칙
        ```python
        (A + B) % p = ((A % p) + (B % p)) % p
        (A * B) % p = ((A % p) * (B % p)) % p
        (A - B) % p = ((A % p) - (B % p) + p) % p
        ```
    * 지수 처리
        ```python
        2 ^ n * 2 ^ n = 2 ^ (2*n)
        ```
